# IOC Search Tool

## Descrição

O IOC Search Tool é uma ferramenta desenvolvida em Python para identificar automaticamente Indicadores de Comprometimento (IOCs) em arquivos de log.
 Utiliza expressões regulares (Regex) para localizar diferentes tipos de IOCs e gera um relatório com os resultados encontrados.

## Funcionalidades

* Percorre automaticamente todos os arquivos.
* Identifica endereços IPv4.
* Identifica URLs.
* Identifica domínios.
* Remove resultados duplicados utilizando `set`.
* Exibe um relatório organizado no terminal.

## Objetivo do Projeto

Este projeto foi desenvolvido com foco em estudos de Segurança da Informação e automação utilizando Python.
